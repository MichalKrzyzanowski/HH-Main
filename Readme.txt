!Important!

If you're going to export the game from a window OS make sure in the model folder is case sensitive.
The model folder is missing files due to it having the same file names. 

To fix this problem follow this link to have case sensitive:
- Enables Windows Features: https://stackoverflow.com/questions/50839623/fsutil-the-request-is-not-supported-after-setcasesensitiveinfo
- Enables Case Sensitivity: https://www.ortussolutions.com/blog/how-to-make-windows-folders-case-sensitive

Download the zip file from the previous release to get the missing files (Folder: HH-Main\RadarGame\Models).
